Minha primeira semana aprendendo HTML

Comecei do zero. Abri o VS Code, digitei meu primeiro <h1> e algo acendeu. Descobri que todo site que já acessei foi construído com as mesmas tags que estou aprendendo agora. Nessa semana entendi a diferença entre estrutura (HTML) e estilo (CSS) — e por que essa separação importa.

Leitura: ~3 min
25/03/2026 Git & Versionamento

Por que aprender Git é indispensável

Antes do Git eu salvava pastas como "projeto-final-v2-revisado-FINAL". Git resolveu isso. Aprendi os comandos básicos: git init, commit, push. O GitHub Pages me permitiu publicar meu site de graça em minutos. Controle de versão não é opcional — é fundamento.

Leitura: ~4 min
15/04/2026 Programação

5 dicas para quem está começando a programar

Depois de meses estudando, aprendi mais com os erros do que com os acertos. Minha lista: (1) pratique todo dia, mesmo que 30 minutos; (2) leia código alheio; (3) não pule a lógica para ir direto ao framework; (4) use o DevTools; (5) documente tudo. Programar é uma habilidade, não talento.

Leitura: ~5 min
