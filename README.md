## 🚀 Como Usar

### 1. **Extrair o Arquivo**
```bash
unzip meu-site.zip
cd meu-site
```

### 2. **Abrir no VS Code**
```bash
code .
```

### 3. **Usar Live Server**
- Instale a extensão "Live Server" no VS Code
- Clique com o botão direito em `index.html`
- Selecione "Open with Live Server"
- O site abrirá automaticamente no navegador

### 4. **Visualizar no Navegador**
- Abra `http://127.0.0.1:5500` (ou a porta mostrada pelo Live Server)

## 🚀 Publicar no GitHub Pages

### Passo 1: Criar repositório
```bash
git init
git add .
git commit -m "Initial commit: Portfolio project"
git branch -M main
git remote add origin https://github.com/MarkoSpsfilho/meu-site.git
git push -u origin main
```

### Passo 2: Ativar GitHub Pages
1. Vá para Configurações do repositório
2. Role até "GitHub Pages"
3. Selecione branch `main` e clique "Save"
4. Seu site estará disponível em: `https://markospsfilho.github.io/meu-site/`
